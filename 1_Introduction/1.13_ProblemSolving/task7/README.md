# Task #7
По данным числам, определите количество чисел, которые равны нулю.  

#### Формат входных данных
Вводится натуральное число N, а затем N чисел.

#### Формат выходных данных
Выведите количество чисел, которые равны нулю.

## Sample Input:
```bash
  5
  1
  8
  100
  0
  12
```

## Sample Output:

```bash
  1
```
