# Task #6

#### Формат входных данных
На вход дается два целых положительных числа a и b.

#### Формат выходных данных
Программа должна вывести среднее арифметическое чисел a и b (ответ может быть целым числом или дробным)

## Sample Input:
```bash
  3 5
```

## Sample Output:

```bash
  4
```

## Sample Input:
```bash
  277 154
```

## Sample Output:

```bash
  215.5
```
